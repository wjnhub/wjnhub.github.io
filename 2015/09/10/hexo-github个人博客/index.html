当提示: No such file or directory 说明你是第一次使用git 此时可创建一个新的ssh秘钥

然后连续3次回车，最终会生成一个文件在用户目录下，打开用户目录，找到.ssh\id_rsa.pub文件 复制里面内容

修改主题

将喜欢的主题下载下来 放到themes文件目录下

修改_config.yml文件中的 theme: landscape 再重新执行hexo g

目前我配置的是next主题 可以打开_config.yml文件进行一些必要配置

再配置中遇到一些小问题

  • menu下的菜单点击后指向报错

    此问题是由于配置中的书写格式存在空格导致的 删除即可(官方示例也是带坑的)

  • categories和tags显示问题

    categories tags和about等默认是被注释的 打开后会报错 是因为缺少这些文件目录

    创建出来即可 文件所在目录为source文件下

    此时再次点击这几个的时候会发现不报错了 可是还是啥内容都没有 这时候我们需要修改下每个目录下的index文件
