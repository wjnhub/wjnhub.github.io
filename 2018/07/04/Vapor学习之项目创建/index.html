生成Xcode项目

使用 Vapor Toolbox 提供的 xcode 命令来生成对应的 Xcode 项目,使其像iOS app开发一样处理

toolbox会询问你是否自动开始xcode,选择y

编译和运行

打开 Xcode 后，选择 scheme 菜单中的 Run scheme，然后点击编译运行按钮。
Xcode终端会显示输出结果

访问 Localhost

打开浏览器输入网址 http://localhost:8080/hello
