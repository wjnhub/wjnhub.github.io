常见问题

问题1：重启终端后 flutter doctor检测显示 zsh: command not found: flutter

解决：

从 macOS Catalina 版开始，您的 Mac 将使用 zsh 作为默认登录 Shell 和交互式 Shell

单次解决

根治 打开.zshrc 添加 source ~/.bash_profile 保存

问题2： 在执行flutter doctor时 一直卡住 Waiting for another flutter command to release the startup lock…

解决：

  1. 关闭Android Studio

  2. 打开flutter安装目录/bin/cache

  3. 删除lockfile文件

  4. 再执行flutter相关命令
