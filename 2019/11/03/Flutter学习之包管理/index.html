简介

Flutter用yaml文件作为其配置文件。Flutter项目默认的配置文件是pubspec.yaml

  • name 应用名
  • description 描述
  • version 应用或包的版本号
  • environment 环境
  • dependencies 应用或包依赖的其它包或插件
  • dev_dependencies 开发环境依赖的工具包,正式包没有
  • flutter flutter 相关的配置选项
